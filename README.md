# Piki's-is2/

Nombre del grupo: Piki's
Nombre del proyecto: Stockeado

En este proyecto vamos a desarrollar un sistema el cual sea capaz de llevar un seguimiento de stock de una ferretería mediana. Y que avise cuando un producto esté por debajo de una cantidad específica de stock, para poder solicitar más de este producto.

Sistema de Gestión de Inventario y Reposición de Stock

<img width="361" height="567" alt="image" src="https://github.com/user-attachments/assets/191cd90d-e99b-4ed9-bb30-d94498c8abe6" />

Este módulo de backend se encarga de la administración centralizada de productos, el control de existencias en tiempo real y la automatización de pedidos a proveedores. El sistema permite registrar cada movimiento de mercadería asociado a un usuario, garantizando la trazabilidad y la generación de alertas automáticas cuando los niveles de stock son críticos.
